# UI UX Pro Max

**מחליף את:** `frontend-design` (deprecated)

**מה זה:** Design intelligence skill שמייצר design systems מותאמים אישית לפרויקט.
מכיל 67 UI styles, 161 color palettes, 57 font pairings, 161 reasoning rules,
support ל-15 tech stacks, ו-pre-delivery checklist.

**מקור:** https://github.com/nextlevelbuilder/ui-ux-pro-max-skill
**הפעלה:** אוטומטית כשמבקשים עבודת UI/UX — אין צורך בslash command.

---

## יכולות עיקריות

- ניתוח אוטומטי של סוג הפרויקט ובחירת design system מתאים
- 24 landing page patterns לפי industry
- 161 "Reasoning Rules" לסקטורים ספציפיים (SaaS, healthcare, eCommerce, …)
- Checklist לפני delivery: accessibility, responsiveness, anti-patterns
- תמיכה ב-React, Vue, Flutter, SwiftUI, ועוד 10 stacks

## יתרון על frontend-design

`frontend-design` (anthropics/skills) מספק multi-pass design process בסיסי.
`ui-ux-pro-max` מוסיף: design system generation, industry rules, delivery checklist,
וcoverage רחב בהרבה של styles ו-palettes.
