## מה כלול בנקודה הזו

- **Skill Orchestration Framework (SIP)** — 7 skill wrappers (superpowers, security-review, graphify, claude-mem, frontend-design, claude-code-workflows, gstack) עם חוזה 4 קבצים לכל אחד
- **`scripts/use-in-project.sh`** — שילוב EOS בפרויקטים אחרים בצורה בטוחה ואידמפוטנטית
- **`scripts/use-engineering-os.command.md`** — פקודת `/use-engineering-os` לסינכרון
- **Graphify L2 mandatory** — גרף נבנה בכל סשן, על כל פרויקט, ללא יוצא מן הכלל
- **`<pattern_gap>`** — נוהל התרעה כשאין תבנית/תיעוד למשימה
- **`core/git-policy.md <cadence>`** — כמה זמן בין קומיטים/ברנץ'ים/מיזוגים
- **`core/documentation-policy.md`** — נוהל תיעוד README
- **`🧰 במה השתמשתי`** — דוח שימוש חובה בסוף כל משימה
